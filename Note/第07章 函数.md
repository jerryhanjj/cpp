### 7.10 函数指针

```c
// 函数指针的声明
double pam(int);
double (*pf)(int);
pf=pam;

// 使用指针来调用函数
double x = pam(4);
double y = (*pf)(4);
double z = pf(4);
```

#### pf 和 (*pf) 等价的原因

1. pf 是函数指针，而 *pf 是函数，因此应用 (*pf)() 调用函数
2. 函数名是指向该函数的指针，指向函数的指针行为应该与函数名相似，应使用 pf()
3. 两种逻辑相冲突，但都被允许
